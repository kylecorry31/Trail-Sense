La herramienta Triangular Ubicación puede usarse para determinar las coordenadas de una ubicación usando dos ubicaciones conocidas. Esto puede servir para determinar tu ubicación si el GPS no funciona, o para determinar las coordenadas de una ubicación en la distancia.

## Determinar tu ubicación
Si tu GPS no funciona y quieres determinar tu ubicación actual, primero tendrás que ver dos ubicaciones conocidas. Idealmente, estas ubicaciones están al menos 60 grados separadas para la mejor precisión. Las ubicaciones pueden ser balizas o puedes determinarlas usando un mapa. Para una guía sobre cómo obtener las coordenadas de una ubicación desde un mapa, consulta la guía 'Mapas Calibrados' en la sección 'Calibrar desde la cuadrícula del mapa'. Una vez que tengas las coordenadas de las dos ubicaciones conocidas, puedes usar los siguientes pasos para determinar tu ubicación:

1. Selecciona 'Mi ubicación' como la ubicación a triangular.
2. Introduce las coordenadas de la primera ubicación conocida y registra la dirección desde donde estás hasta esa ubicación como Ubicación 1. Puedes pulsar el icono de la cámara en el lado derecho del campo de dirección para apuntar la ubicación usando la cámara.
3. Repite para Ubicación 2 usando la segunda ubicación conocida.
4. Si tu ubicación puede determinarse, se mostrará en la parte superior de la pantalla.

## Determinar la ubicación de un destino
Si puedes ver un destino desde múltiples ubicaciones, puedes usar la herramienta Triangular Ubicación para determinar las coordenadas del destino.

1. Selecciona 'Destino' como la ubicación a triangular.
2. Registra tu ubicación y registra la dirección desde donde estás hasta esa ubicación como Ubicación 1. Puedes pulsar el icono de GPS junto al campo de ubicación para usar tu ubicación actual. Puedes pulsar el icono de la cámara en el lado derecho del campo de dirección para apuntar la ubicación usando la cámara.
3. Muévete a una nueva ubicación que esté al menos a 30 m / 100 ft de la primera ubicación y repite para Ubicación 2. Puedes guardar el teléfono mientras te mueves; la herramienta guardará los datos introducidos.
4. Si el destino puede determinarse, se mostrará en la parte superior de la pantalla.

## Mapa de triangulación
A medida que introduces ubicaciones, el mapa en la parte superior de la pantalla se actualizará para mostrar la intersección de las dos líneas. Puedes usar esto para verificar que las ubicaciones que has proporcionado son correctas. El mapa siempre tiene el Norte en la parte superior.

## Opciones para compartir
Una vez que triangules una ubicación, tienes varias opciones:

- **Baliza:** Crea una baliza en la ubicación triangulada. Esto abrirá la herramienta Beacons. Para más información sobre balizas, consulta la guía 'Beacons'.
- **Navegar:** Navega a la ubicación triangulada. Esto abrirá la herramienta Navigation. Para más información sobre Navegar, consulta la guía 'Navigation'.
- **Compartir...:** Comparte la ubicación triangulada. Esto abrirá el menú de compartir, donde puedes copiar las coordenadas, compartir como código QR, abrir en una app de mapas, o compartir con otra app (p. ej., un mensaje de texto).

## Reiniciar
Puedes restablecer la herramienta pulsando el botón de restablecer en la parte inferior de la pantalla.
