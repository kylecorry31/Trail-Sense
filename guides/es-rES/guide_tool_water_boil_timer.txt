La herramienta Temporizador de Hervir Agua te ayuda a determinar el tiempo de ebullición necesario para que el agua sea segura para beber, teniendo en cuenta tu elevación actual, ya que el tiempo de ebullición varía con la elevación.

## Hacer que el agua sea segura para beber
Para asegurarte de que el agua sea segura para consumir, primero debes filtrar y luego hervirla. Ten en cuenta que filtrar y hervir no hará que el agua contaminada sea segura para beber. Sigue estos pasos para que el agua sea potable:

1. Filtra el agua en un recipiente metálico.
2. Lleva el agua a ebullición vigorosa.
3. Pulsa el botón 'Iniciar' una vez que el agua alcance la ebullición vigorosa.
4. Es probable que el agua sea segura para beber cuando el temporizador expire.

Por defecto, el temporizador usa tu elevación actual para calcular el tiempo de ebullición requerido. Alternativamente, puedes seleccionar manualmente una duración de 1 minuto o 3 minutos usando los botones situados en la parte inferior de la pantalla.

Puedes elegir usar la alarma para este temporizador activándola en 'Ajustes > Temporizador de Hervir Agua > Alarma para temporizador de hervido'. Se reproducirá un sonido de alarma aunque los medios y notificaciones estén silenciados.

## Descargo de responsabilidad
Esta herramienta usa recomendaciones de organizaciones sanitarias, pero si se hace incorrectamente o si el agua está contaminada aún podrías ponerte enfermo. Úsala bajo tu responsabilidad.
