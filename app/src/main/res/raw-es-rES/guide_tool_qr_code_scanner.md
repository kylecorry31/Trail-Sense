La herramienta Escáner de códigos QR puede usarse para escanear códigos QR.

## Escanear un código QR
Para escanear un código QR, simplemente apunta tu cámara al código. Una vez reconocido, el contenido se mostrará en pantalla junto con algunas acciones que puedes realizar.

### Ubicación
Si el código QR contiene una ubicación, tendrás las siguientes opciones:
- **Baliza**: Crea una baliza usando la ubicación. Esto abrirá la herramienta Balizas con la ubicación rellenada.
- **Mapas**: Abre la ubicación en una app de mapas de tu teléfono.
- **Copiar**: Copia el contenido del código QR al portapapeles.
- **Eliminar**: Elimina el resultado de la lista.

### URL
Si el código QR contiene una URL, tendrás las siguientes opciones:
- **Open**: Abre la URL en el navegador predeterminado.
- **Copiar**: Copia el contenido del código QR al portapapeles.
- **Eliminar**: Elimina el resultado de la lista.

### Texto
Si el código QR contiene texto o es un tipo no compatible, tendrás las siguientes opciones:
- **Nota**: Crea una nota con el texto y mostrará un banner en la parte inferior de la pantalla con un botón para acceder a la nota.
- **Copiar**: Copia el contenido del código QR al portapapeles.
- **Eliminar**: Elimina el resultado de la lista.

## Compartir datos con códigos QR
Algunas herramientas soportan el intercambio de datos mediante códigos QR. Para usar esta función, sigue las instrucciones en la guía de la herramienta específica para generar el código y escanéalo con la herramienta Escáner de códigos QR. También puedes escanear una baliza directamente desde la herramienta Balizas.

Las herramientas que soportan intercambio de datos son:
- Balizas
- Notas
- Navegar (tu ubicación)
