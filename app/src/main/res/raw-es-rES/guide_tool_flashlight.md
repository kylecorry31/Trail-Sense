La herramienta Linterna puede usarse para proporcionar luz o hacer señales para pedir ayuda. Como con cualquier herramienta en Trail Sense, siempre es mejor llevar una linterna real contigo.

## Linterna
Pulsa el botón de la linterna para alternar la linterna encendida o apagada. Cuando la linterna esté encendida (en cualquier modo), se mostrará una notificación para permitirte apagarla rápidamente.

En dispositivos compatibles, puedes ajustar el brillo usando el deslizador en la parte inferior.

## Estroboscópica
Arrastra el dial hasta la frecuencia estroboscopica deseada. Pulsa el botón de la linterna para alternar entre estroboscópica encendida o apagada.

Ten en cuenta que la función estroboscópica contiene luces intermitentes que pueden ser peligrosas para algunos usuarios. El uso prolongado de la estroboscópica puede provocar un gasto excesivo de batería o dañar el flash de la cámara.

## SOS
Arrastra el dial hasta el indicador SOS para reproducir continuamente la señal de socorro SOS (3 cortas, 3 largas, 3 cortas). Pulsa el botón de la linterna para alternar el SOS entre encendido o apagado.

## Pantalla
Puedes usar la pantalla como linterna, lo que puede ser mejor para trabajos de cerca. Pulsa el botón de pantalla para iniciar. A partir de ahí puedes ajustar el brillo usando el deslizador en la parte inferior y alternar el color entre rojo (preserva la visión nocturna) y blanco (más brillante) usando el botón en la parte superior.

Pulsa atrás o pulsa el botón 'Apagar' para apagar la linterna de pantalla. Tus ajustes se guardarán para la próxima vez.

Puedes elegir controlar el brillo de la linterna de pantalla habilitando 'Ajustes > Linterna > Controlar brillo de pantalla con botones de volumen'.

## Apagado automático
Puedes configurar la linterna para que se apague automáticamente tras un cierto tiempo habilitando 'Ajustes > Linterna > Apagado automático' y seleccionando el tiempo deseado. Esto puede ser útil para ahorrar batería si olvidas apagar la linterna.

## Alternar con botones de volumen
Puedes configurar la linterna para alternar encendida o apagada cuando pulses cualquiera de los botones de volumen habilitando 'ajustes > Linterna > Alternar con botones de volumen'. Sólo funciona mientras Trail Sense esté abierto.

## Acción rápida

### Linterna
Puedes habilitar la acción rápida Linterna en los ajustes para la pestaña en la que la quieras.

Para usar la acción rápida, pulsa el botón de acción rápida de la linterna para alternarla encendida o apagada.

### Linterna de pantalla
Puedes habilitar la acción rápida Linterna de pantalla en los ajustes para la pestaña en la que la quieras. Pulsa el botón de acción rápida para abrir la linterna de pantalla.

## Widgets
Los siguientes widgets pueden colocarse en la pantalla de inicio del dispositivo o verse en la app:

- **Linterna**: Alterna entre linterna encendida o apagada.
