La herramienta Reloj se puede usar para determinar la hora actual usando los satélites GPS.

## Ver la hora GPS
Cuando abres por primera vez la herramienta Reloj se muestra la hora configurada en tu dispositivo. Una vez que obtienes el posicionamiento GPS, el reloj se actualiza automáticamente para mostrar la hora GPS precisa. Para actualizar la hora GPS, pulsa el icono de actualizar en la esquina superior derecha.

## Sincronizar el reloj
Para sincronizar el reloj de tu dispositivo con la hora GPS:

1. Pulsa en 'Actualizar la hora del sistema'.
2. Anota la hora que aparece en el cuadro emergente.
3. Pulsa 'OK' para abrir los ajustes de hora del dispositivo.
4. Introduce la hora anotada y espera la notificación de sincronización antes de guardar.
5. Guarda la actualización de la hora cuando llegue la notificación.
6. La hora del sistema ya está sincronizada con la hora GPS.

## Reloj digital
Si prefieres un reloj digital, puedes cambiar a este deshabilitando la opción 'Reloj analógico' en 'Ajustes > Herramientas > Reloj'. Se mostrará un reloj digital grande en la pantalla.
