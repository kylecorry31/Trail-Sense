La herramienta Notas puede usarse para escribir notas para acceso sin conexión.

## Crear o editar una nota
Para crear, ver o editar una nota:

1. Pulsa el botón '+' en la esquina inferior derecha para crear una nota, o pulsa sobre una existente en la lista.
2. Puedes editar el título y/o el contenido de la nota.
3. Pulsa el botón de marca de verificación en la esquina inferior derecha para guardar.

## Eliminar una nota
Para eliminar una nota:

1. Pulsa el icono de menú en el lado derecho de la nota en la lista.
2. Pulsa 'Eliminar'.

## Compartir una nota (código QR)
Para compartir una nota como código QR:

1. Pulsa el icono de menú en el lado derecho de la nota en la lista.
2. Pulsa 'Código QR '.
3. Escanea el código QR con otro dispositivo para importar la nota, o haz una captura de pantalla para usarla más tarde.

Nota: El código QR solo puede transferir los primeros 1000 caracteres de una nota.

## Acción rápida
Puedes habilitar la acción rápida Nota en los ajustes para la pestaña deseada.

Para usar la acción rápida, pulsa el botón de acción rápida Nota para crear una nueva nota.
