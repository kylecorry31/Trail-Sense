La herramienta Listas de Equipaje puede usarse para planificar un viaje y recordar empaquetar todos los elementos necesarios.

## Crear una lista de equipaje
Para crear una nueva lista de equipaje:

1. Pulsa el botón '+' en la esquina inferior derecha.
2. Selecciona 'Nueva lista de equipaje'.
3. Introduce un nombre para la lista de equipaje.

## Copiar una lista de equipaje
Para copiar una lista de equipaje existente:

1. Pulsa el botón de menú a la derecha de una lista de equipaje existente.
2. Pulsa "Copiar".
3. Introduce un nombre para la nueva lista de equipaje.
4. (Opcional) Pulsa el botón de menú en la parte superior derecha y pulsa "Eliminar cantidades" para restablecer las cantidades empaquetadas.

## Añadir elementos

Con una lista de equipaje abierta, pulsa el botón '+' en la esquina inferior derecha. En la pantalla de crear elemento, rellena los campos como se describe a continuación:

- **Nombre**: El nombre del elemento
- **Cantidad empaquetada**: El número de elementos actualmente empaquetados; puede estar en blanco o un número (con decimales)
- **Cantidad deseada**: El número de elementos que quieres empaquetar; puede estar en blanco o un número (con decimales)
- **Categoría**: La categoría para agrupar este elemento; por defecto es "Otro"
- **Peso**: El peso de 1 elemento en tus unidades deseadas. Trail Sense escalará esto por la cantidad de elementos empaquetados para determinar el peso total.

Cuando termines, pulsa el botón de verificación en la esquina inferior derecha para guardar el elemento.

## Editar elementos

Con una lista de equipaje abierta, pulsa el elemento que quieres editar. Esto te llevará al mismo formulario descrito en la sección 'Añadir elementos'.

Cuando termines, pulsa el botón de verificación en la esquina inferior derecha para guardar el elemento.

## Marcar elementos como empaquetados
Si has empaquetado un elemento, puedes editarlo o pulsar la casilla junto a él para marcarlo como empaquetado. Esto establecerá la cantidad empaquetada igual a la cantidad deseada y actualizará el peso (si procede).

## Peso de la mochila
Si los elementos tienen peso asociado, el peso total de la mochila se mostrará en la parte superior de la pantalla. Las unidades del peso total pueden cambiarse en 'Ajustes > Unidades > Peso'.

## Ordenar una lista de equipaje

Pulsar el menú en la parte superior derecha de la lista de equipaje te permitirá ordenar tus elementos.

- **Categoría**: Agrupa elementos por categoría y los ordena por nombre dentro de la categoría.
- **Porcentaje empaquetado (bajo a alto)**: Los elementos sin empaquetar se muestran arriba, mientras que los empaquetados bajan al final. Usalo para ver más fácilmente lo que queda por empaquetar.
- **Porcentaje empaquetado (alto a bajo)**: Los elementos empaquetados se muestran arriba, mientras que los no empaquetados bajan al final. Usalo para ver más fácilmente lo que ya está empaquetado.
- **Peso (bajo a alto)**: Los elementos que ocupan menos peso en la mochila se ordenan arriba.
- **Peso (alto a bajo)**: Los elementos que ocupan más peso en la mochila se ordenan arriba. Úsalo para ver qué elementos lastran más tu mochila.

## Eliminar una lista de equipaje

Puedes eliminar una lista de equipaje pulsando el botón de menú a la derecha de la lista de equipaje y pulsando 'Eliminar'*. Esto eliminará permanentemente la lista de equipaje y todos los elementos que contiene.

También puedes eliminar desde la propia lista de equipaje pulsando el botón de menú en la parte superior derecha y pulsando 'Eliminar'.

## Renombrar una lista de equipaje

Puedes renombrar una lista de equipaje pulsando el botón de menú a la derecha de la lista de equipaje y pulsando 'Renombrar'. Esto te permitirá introducir un nuevo nombre para la lista de equipaje.

También puedes renombrar desde la propia lista de equipaje pulsando el botón de menú en la parte superior derecha y pulsando 'Renombrar'.

## Eliminar cantidades empaquetadas

Puedes eliminar las cantidades empaquetadas de todos los elementos de una lista de equipaje pulsando el botón de menú a la derecha de la lista de equipaje y pulsando 'Eliminar cantidades'. Esto establecerá todas las cantidades empaquetadas a 0, pero no eliminará los elementos.

## Exportar una lista de equipaje
Puedes exportar una lista de equipaje pulsando el botón de menú a la derecha de la lista de equipaje y pulsando 'Exportar'. Se te solicitará guardar un archivo .csv en tu dispositivo. Este archivo puede ser importado por Trail Sense o usado en el sitio web 'LighterPack'.

## Importar una lista de equipaje
Puedes importar una lista de equipaje pulsando el botón '+' en la esquina inferior derecha y pulsando 'Import packing list'. Se te solicitará seleccionar un archivo .csv de tu dispositivo. El archivo csv debe estar en el formato exportado por Trail Sense o LighterPack.

Se te pedirá introducir un nombre para la lista de equipaje recién importada. Si la lista tenía cantidades empaquetadas previamente, se importarán, pero puedes eliminarlas siguiendo los pasos de la sección 'Eliminar cantidades empaquetadas'.
